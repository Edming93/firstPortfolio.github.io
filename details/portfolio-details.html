초심자를 위한 스타터페이지

  • 1. Main페이지 실행 전 interceptor를 태워 session이 없는 사용자에게 보여지는 가이드 화면 제공.
  • 2. svg 벡터 이미지를 사용하여 path 선을 연결해 화면이 스크롤 되어 내려갈 때마다 그려지는 효과 적용. strokeDasharray, strokeDashoffset 이용
  • 3. 로그인 성공 시 session에 유저 정보를 저장. 세션이 있는 사용자는 로고를 클릭해 indexPage로 이동시 스타터 페이지 비활성화. 로그아웃 시 세션 정보가 초기화 되면서 스타터페이지 재 활성화
  • 4. 로고에 scrollTop을 이용하여 스크롤되어 배경색이 바뀔 때 마다 글자색을 바꿔주는 효과 적용
  • 5. @media를 이용한 반응형 구현
  • Project URL : 깃링크
